# View Apex Detailing Business Card (V2)

Cloud Agent files are **not on your computer** until you download them.
Do **not** use `https://workspace/...` — that URL does not exist.

## Option 1 — Open image in browser (fastest)

Click or paste these links:

- **Combined preview:** https://raw.githubusercontent.com/Apex-Web-Worx/apex-detailing-website/cursor/apex-business-card-931b/attached_assets/apex-business-card-v2-preview.png
- **Front:** https://raw.githubusercontent.com/Apex-Web-Worx/apex-detailing-website/cursor/apex-business-card-931b/attached_assets/apex-business-card-v2-front.png
- **Back:** https://raw.githubusercontent.com/Apex-Web-Worx/apex-detailing-website/cursor/apex-business-card-931b/attached_assets/apex-business-card-v2-back.png

## Option 2 — Download from GitHub website

1. Open: https://github.com/Apex-Web-Worx/apex-detailing-website/tree/cursor/apex-business-card-931b/attached_assets
2. Click `apex-business-card-v2-preview.png`
3. Click **Download** (or right-click image → Save As)

## Option 3 — Pull to your computer

```bash
git fetch origin cursor/apex-business-card-931b
git checkout cursor/apex-business-card-931b
open attached_assets/apex-business-card-v2-preview.png
```

(Mac: `open` · Windows: `start` · Linux: `xdg-open`)
